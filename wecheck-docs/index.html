WeCheck your Code!

Let us help you find out how you can improve your code.

The first service we provide is calculating the percentage of javadoc in your github project.

Set Up

In order to start using WeCheck you need to follow these simple steps:

  1. Add this url as a Webhook in your github repo: https://api-wecheck.wedeploy.io/github/pull-request
  2. Give the user liferay-dev-ci permission to write in your repo.
  3. Add a file in the root of your github repo called: wecheck.properties.
    This file should contain the following properties:
    Property Value
    include.directories The list of directories that should be scanned (separated by commas)
    exclude.directories (optional) The list of directories that shouldn't be scanned (separated by commas)
    report.title (optional) The name of the report that is generated
